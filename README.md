# UniLink

UniLink is a modern social networking platform designed for university students to connect, share, and collaborate.

## Project Structure

- **backend/**: Express.js server with MongoDB using Mongoose.
- **frontend/**: React application built with Vite and Tailwind CSS.

## Features

- **Authentication**: Secure login and registration with JWT.
- **Posts**: Create, view, like, and comment on posts.
- **Profiles**: Personalized user profiles with follow/unfollow functionality.
- **Groups & Events**: Collaboration spaces for student groups and upcoming university events.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- MongoDB

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/knockknock10/Unilink.git
   ```

2. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```

3. Install frontend dependencies:
   ```bash
   cd ../frontend
   npm install
   ```

### Running the App

1. Start the backend:
   ```bash
   cd backend
   npm start
   ```

2. Start the frontend:
   ```bash
   cd ../frontend
   npm run dev
   ```
